== Description ==

Adds a Facebook 'Like' button to any page on your blog

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `jp-facebook.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use [JP-Facebook-Like] shortcode to on your page or post
1. See http://www.jpreece.com/tutorials/wordpress/jp-facebook/ for complete tutorial
